## Temas Investigados para Mañana

### 1. Arquitectura de Bots de Mensajería
**Concepto clave:** Un bot de WhatsApp/Instagram necesita:
- **Webhook**: URL que recibe mensajes cuando alguien escribe
- **Procesador**: Código que analiza el mensaje y decide qué responder
- **API de envío**: Para enviar respuestas de vuelta

### 2. Opciones para Practicar (sin costo)

**Sandbox de Meta (Gratis):**
- Permite probar WhatsApp Business API sin pagar
- Limitado a 5 números de teléfono de prueba
- Perfecto para aprender

**Simuladores locales:**
- Podemos crear una "web app" que simule WhatsApp
- Te permite ver cómo funciona sin depender de APIs externas
