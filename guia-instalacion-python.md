# Guía de Instalación de Python para Andrés

## Paso 1: Descargar Python

1. Abre tu navegador
2. Ve a: https://www.python.org/downloads/
3. Haz clic en el botón amarillo grande que dice "Download Python 3.12.x"
4. Se descargará un archivo .exe

## Paso 2: Instalar Python (MUY IMPORTANTE)

1. Abre el archivo descargado
2. **⚠️ MARCAR LA CASILLA:** "Add Python 3.12 to PATH" (es una casilla en la parte inferior de la primera pantalla)
3. Haz clic en "Install Now"
4. Espera a que termine
5. Haz clic en "Close"

## Paso 3: Verificar Instalación

1. Presiona `Windows + R`
2. Escribe `cmd` y presiona Enter
3. En la ventana negra que aparece, escribe:
   ```
   python --version
   ```
4. Debe aparecer algo como: `Python 3.12.x`

## Paso 4: Verificar pip (gestor de paquetes)

En la misma ventana de comandos, escribe:
```
pip --version
```
Debe aparecer la versión de pip.

## ¿Problemas?

Si algo falla, tomamos screenshot y lo resolvemos juntos.
